# System Design Projects

A collection of system design projects implemented in Rust. This repository serves as a personal learning space to explore backend architecture, system design principles, and Rust programming.

**Disclaimer:** This project is strictly for educational and learning purposes. It is **not** intended for, nor should it be used in, production environments.

## Projects

* **[01 URL Shortener](./projects/01_url_shortener):** A URL shortening service.
* **[02 Prospect Web Crawler](./projects/02_prospect_web_crawler):** A web crawler engine.
* **[03 Kafka-lite](./projects/03_kafka_lite):** A lightweight message broker.

## Structure

* `common/`: Shared libraries and utilities used across multiple projects.
* `projects/`: Individual system design implementations.
